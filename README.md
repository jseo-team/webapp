
# Install
```bash
yarn install
set NODE_ENV=production yarn build
sudo yarn link
sudo yarn link "jseo"
```

To add user <username>
```
jseo add <username>
```

# Configure
```bash
set JSEO_HOME=~/.jseo
jseo start
```



# Compile / Build
```bash
yarn build
```





# Server start
From source folder
```bash
yarn start
```
(Anywhere) 
```bash
jseo start
```
